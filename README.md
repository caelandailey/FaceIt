# FaceIt
Reason for this repo is to learn about UIBezierPath and how to draw lines.
In this project you may adjust the follow properties.

- mouthCurvature -> 1.0 is full smile and -1.0 is full frown
- eyesOpen
- scale 
- lineWidth
- color
